# IT Homelab Infrastructure

This repository documents a hands-on homelab environment used to develop skills in enterprise IT infrastructure and system administration.

## Lab Environment

The lab environment simulates a small enterprise network using virtual machines.

Core systems:

DC01 – Windows Server Domain Controller  
CLIENT01 – Windows 11 workstation  
LINUX01 – Ubuntu Server

## Technologies Practiced

Windows Server  
Active Directory  
DNS  
Group Policy  
Linux administration  
SSH remote management  
File sharing and access control  

## Lab Projects

01 - Virtualization Setup  
02 - Active Directory Domain Controller  
03 - Domain Client Join  
04 - File Share Permissions  
05 - Linux Server Administration

Each lab contains documentation, configuration notes, and screenshots demonstrating successful deployment and testing.
