# Domain Client Join

This lab documents the process of joining a Windows client system to an Active Directory domain.

A Windows 11 virtual machine was configured to communicate with the domain controller and authenticate using domain credentials.

Key tasks performed:

- Configure DNS to point to the domain controller
- Join the workstation to the Active Directory domain
- Verify domain authentication
- Log in using domain user accounts

Skills practiced:

- Windows workstation configuration
- Active Directory domain authentication
- DNS troubleshooting
- enterprise client management

This lab simulates the process used in corporate IT environments when deploying or provisioning employee workstations.
