# File Share Permissions

This lab focuses on configuring shared network storage within an Active Directory environment.

A shared folder was created on the Windows Server domain controller and access was controlled using Active Directory users and groups.

Key tasks performed:

- Create shared network folder
- Configure share permissions
- Configure NTFS permissions
- Test access from a domain joined client
- Verify user level access control

Skills practiced:

- Windows Server file sharing
- permission management
- access control using Active Directory groups
- troubleshooting file access issues

This lab simulates common IT administrative tasks related to network file storage and user access management.
