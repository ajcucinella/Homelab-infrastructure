# Linux Server Administration

This lab focuses on basic Linux server management and remote administration.

An Ubuntu Server virtual machine was deployed and configured to allow secure remote management through SSH.

Key tasks performed:

- Install Ubuntu Server
- Configure network settings
- Enable and test SSH access
- Create and manage Linux users
- Install and manage system packages
- Monitor system services

Skills practiced:

- Linux command line administration
- SSH remote access
- user and permission management
- service monitoring and troubleshooting

This lab introduces core Linux administration tasks commonly used in cloud infrastructure and enterprise server environments.
